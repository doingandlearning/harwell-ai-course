# Module 8 — Demos

## Demo 1: Reasoning model example (optional)

**Purpose:** Show how reasoning models work differently from standard models.

**Materials:**
- Access to reasoning model (if available) or screenshots
- Example showing step-by-step reasoning

**Setup:**
- Only if you have access to reasoning model
- Prepare example showing reasoning process

**Steps:**
1. Show standard model response (no reasoning shown)
2. Show reasoning model response (with step-by-step reasoning)
3. Compare: Standard vs. reasoning
4. Discuss: Benefits and implications

**Key points:**
- Reasoning models show their thinking process
- More transparent, can verify reasoning
- Better for complex tasks
- Still need human oversight

**Note:** Only run if you have access. Otherwise, use slides or skip.

## Presenter notes

- **Keep it forward-looking** — focus on trends and next steps
- **Balance optimism with realism** — AI is powerful but has limitations
- **Emphasize fundamentals** — prompting, review, testing don't change
- **Make next steps concrete** — help delegates create actionable plans
- **End on empowering note** — they have the skills and a plan
- **Tie back to "what must feel different on Monday"** — ensure delegates have concrete answer
